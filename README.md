### Getting Started

Clone this repo:
```
git clone https://github.com/talitta-nunes/zipcode-tracker.git
```

Install the required dependencies:
```
pip install -r requirements.txt
```

Run the server:
```
python app.py
```

Access the url:
```
http://127.0.0.1:5000
```
